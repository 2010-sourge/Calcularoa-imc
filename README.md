# 🧮 Calculadora de IMC

> **Índice de Massa Corporal** — Uma aplicação web simples para calcular e interpretar o IMC do usuário.

---

## 📋 Sobre o Projeto

A **Calculadora de IMC** é uma aplicação web que permite ao usuário inserir seu peso e altura para calcular o Índice de Massa Corporal (IMC), utilizando a fórmula padrão da OMS.

---

## 🚀 Funcionalidades

- Entrada de peso em **kg**
- Entrada de altura em **cm**
- Cálculo automático do IMC
- Exibição da fórmula utilizada: `IMC = Peso (kg) ÷ Altura² (m)`

## 📐 Fórmula do IMC

```
IMC = Peso (kg) ÷ Altura² (m)
```

### Tabela de Classificação (OMS)

| IMC           | Classificação         |
|---------------|-----------------------|
| Abaixo de 18,5 | Abaixo do peso       |
| 18,5 – 24,9   | Peso normal           |
| 25,0 – 29,9   | Sobrepeso             |
| 30,0 – 34,9   | Obesidade Grau I      |
| 35,0 – 39,9   | Obesidade Grau II     |
| Acima de 40,0  | Obesidade Grau III   |
